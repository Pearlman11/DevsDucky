# VR Maze Locomotion Demo

A Unity VR project demonstrating various locomotion techniques for Meta Quest headsets, featuring arm swing movement, teleportation, and interactive object manipulation in a maze environment.

## Features

### Locomotion Systems
- **Arm Swing Movement** - Natural walking simulation by swinging your arms
- **Teleportation** - Point and teleport to valid surfaces
- **Snap Rotation** - Quick 15Â° turns using thumbstick
- **Go-Go Technique** - Extended reach for distant objects
- **Air Grab** - Pull yourself through space by grabbing air
